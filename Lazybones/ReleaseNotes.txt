v.1.0.2
   - Background update checks now run on every sit/stand cycle start (throttled to at most once per 30 minutes), so long-running sessions pick up new releases without restarting the app.
   - The daily goal is now a number of completed standing cycles rather than a number of standing minutes. The settings field reads "Daily standing cycles"; the outer ring and Today stat show cycle progress (with the minutes-stood-today shown as a secondary number). A cycle only counts if it ran to natural completion — bailing out via Swap doesn't count toward the goal, though the time you did spend standing is still tracked.
   - Streak follows the new cycle goal: a streak day is an active day that hit the cycle count. Days where the screen was locked all day or the app was paused are "paused" — neither hit nor miss. No mid-day instant-loss action; the streak fails at end-of-day if an active day finishes below the goal.
   - Old saved settings using the previous minutes-based goal are ignored; the new daily-cycle goal starts at its default. Lifetime achievements remain minutes-honest.
   - Stats tab: the 13-week minutes heatmap is now half-width, balanced by a new 14-day cycles-per-day bar chart on the right. The main-disk dashboard button switched from a cog to a stats glyph so the contents are less hidden.

v.1.0.1
   - Added an Updates tab to the dashboard showing the running version, status, release notes for any new version, and Check-for-updates / Restart-now buttons.
   - New blue indicator below the clock on the main disk when an update is ready — click to jump to the Updates tab.
   - Clicking the cog button now restores the dashboard when it's been minimized via the taskbar instead of doing nothing.
   - Removed the dashboard's minimize button — closing is the right gesture for a transient dialog.
   - Release pages now include an install guide explaining the SmartScreen / Gatekeeper warnings and the click-through steps for each platform.
   - Fixed: release-notes extraction in CI no longer drops the body when the file contains only one version entry.

v.1.0.0
   - First public release of Get Up, Lazybones!
   - Sit/stand reminder that lives as a round transparent desktop disk on Windows and macOS.
   - Layered progress rings track today's standing goal (outer) and the current cycle (inner).
   - Auto-pauses when you lock your screen and resumes on unlock — pause time is invisible to every metric.
   - Achievements covering streaks, daily intensity, response engagement, and lifetime volume.
   - Dashboard with stats, achievements, settings, and an updates tab.
   - Optional "Start at login" on both platforms (Windows Run key / macOS LaunchAgent).
   - Auto-update from GitHub Releases via Velopack — new versions show as a blue indicator on the main disk and a dedicated Updates tab.
